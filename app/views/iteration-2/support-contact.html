If you have an issue with the tool or need access, you must raise a service request using HaloITSM.

  • access requests

Submit a service request via HaloITSM (opens in new tab)

When submitting your request, include:

  • your name and organisation
  • a description of the issue or request
  • screenshots or error messages (if available)
  • the date and time the issue occurred

This helps us resolve your request more quickly.
