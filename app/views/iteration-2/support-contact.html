Report a problem or request access

If you are having an issue with the tool or need access, you must raise a service request using HaloITSM.

Use HaloITSM for:

  • add or remove access
  • access requests
  • technical problems using the tool
  • data or functionality issues
  • general support queries

Submit a service request via HaloITSM (opens in a new tab)

When submitting your request, include:

  • your name and organisation
  • a description of the issue or request
  • screenshots or error messages (if available)
  • the date and time the issue occurred

This will help us resolve your request more quickly.
