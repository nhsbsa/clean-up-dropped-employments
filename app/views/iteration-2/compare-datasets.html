Compare data sets

Learn how you can compare two data sets side by side using the Identify record issues tool

From the results screen of the Identify record issues tool if you scroll down, you can see the compare data sets side by side. You can compare any two data sets you like. Click apply and your table will be adjusted accordingly. You can do this with both sides. And this just makes it easier to read if you're looking for specific data and then you can click back to go back to the main view.

[00:57]
Another way of selecting datasets to compare, if you're looking at a data set and you think you would like to compare it, you can just click compare dataset here and it will automatically select it and then you can choose which one you would like to compare it with.
