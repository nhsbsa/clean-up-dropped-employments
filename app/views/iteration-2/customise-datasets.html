Customise data sets

Learn how you can customise data sets using the Identify record issues tool

{% call details({ summaryText: "View the video transcript" }) %}

[00:01]
From the results screen of the Identify records issues tool, you can scroll down and within each dataset you will see there is a customise dataset button. From here you can click this and select any columns that you would like to remove just to make your view easier to read. If you are only looking at the end date and the start date and the employee code, you could unselect the rest of the columns, click apply and your table would become much clearer and easier to read. If you want to reinstate the data, you can just reselect.

[00:42]
Click apply and your table will go back to how it was originally. This is visible within every dataset shown on the tool.
