Filter data sets

Learn how you can filter data sets using the Identify record issues tool

{% call details({ summaryText: "View the video transcript" }) %}

[00:01]
On the results screen of the Identify Record Issues tool you will see a show filters button. This lists all the data sets that are currently included in the tool. You will see two of them are greyed out here. This is because these contain issues. Any data set containing an issue cannot be hidden using the filter.

[00:24]
The others can be. So to hide a data set you would just click the ones you wanted to hide. So if you wanted to hide all of them apart from those with issues, just uncheck them, apply filters and then you would see there's only the two data sets with issues found which are left, equally, to turn them back on, you just reselect them and click apply filters and all data sets would reappear. You can turn on and off as many as you want at one time.
