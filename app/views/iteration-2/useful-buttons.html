Search new member

Navigate from one member to another

{% call details({ summaryText: "View the video transcript" }) %}

[00:01]
From the results screen of the Identify record issues tool, you will see the search new member button in the top right. You can click this at any point to go back to the search screen and search for a new member.

Update data sets

Update data being pulled from Compendia

{% call details({ summaryText: "View the video transcript" }) %}

[00:01]
From the results screen of the Identify record issues tool, you can scroll down and you will see update data sets button. You may have made some changes and want to re-check on the tool so you can just click update datasets and this will refresh your screen and the data will be re pulled from Compendia so you know it is the latest data being shown.

{% endcall %}

Hide correct rows

Hide correct rows appears on every data set which contains an issue

{% call details({ summaryText: "View the video transcript" }) %}

[00:01]
From the results screen of the Identify record issues tool. When looking at a data set which contains an issue, you can use the Hide Correct Rows button to hide all rows apart from those with issues. You can click it again and the rows will reappear.

{% endcall %}
