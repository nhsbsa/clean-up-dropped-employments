Features and concept testing:

HaloITSM request - PaPPs
DESIGN UR

Design

This prototype has been created to explore the option of creating a HaloITSM ticket through the tool, giving the user the option of creating a request from scratch using the menu link from the search page or the pension issues overview or they can create a prepopulated request form from within a members record; using the menu item or the section at the bottom of the screen below the last data set.

Users are able to edit the form as required, once the user continues after filling in the form they can check their answers before submitting to service desk.
